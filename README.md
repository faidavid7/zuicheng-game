# 罪城法則 — 荷官主控台

課堂桌遊「罪城法則」的荷官（遊戲主持人）電腦主控台：追蹤 20 格環狀盤面上所有玩家的位置、監獄、逃脫券、皮斯、感染值與名次，自動偵測隱藏個人規則與團體規則的觸發時機，骰子結果由荷官手動輸入。單一 HTML 檔、純 vanilla JS、完全離線可用、零外部依賴。

## 線上使用

直接開網站：**<https://faidavid7.github.io/zuicheng-game/>**

進度自動存在瀏覽器 localStorage（每台裝置各自獨立）；換裝置請用畫面右上「⬇ 匯出 / ⬆ 匯入」搬移存檔 JSON。

## 下載

不想連網也可以離線用：到本頁按綠色 `Code` → `Download ZIP` 下載整個專案（或直接另存 `index.html`）。

## 解壓縮

把下載的 `.zip` 解壓縮到你想放的位置，會得到一個 `zuicheng-game/` 資料夾。

## 選擇資料夾

打開解壓縮出來的資料夾，雙擊其中的 `index.html`，用任何現代瀏覽器（Chrome / Edge / Firefox / Safari）開啟即可，不需安裝、不需網路。

## 用法

1. **登記**：📝 逐位玩家選五色字塊組成隱藏個人規則（或 📋 直接貼上 Excel 登記表，支援「擲骰子順序」欄自動排序）
2. **開始**：▶ 開始遊戲後，每回合點選實際擲出的骰面（普通骰 1/2/2/3/監獄/逃脫、特殊骰 1/2/2/3/3/3）
3. **裁決**：規則觸發時會跳出建議卡，荷官按「執行 / 略過」；皇宮三選一、高岡樹下二選一、團體規則可隨時多條增修
4. **調整**：任何現實中的狀況都可直接改——每位玩家有 ✏️ 編輯視窗、指定擲骰者下拉、全員倒退兩格特殊鈕；誤操作一鍵 ↩ 復原
5. 遊戲無自動終止條件；過終點只記名次，可一直玩下去
